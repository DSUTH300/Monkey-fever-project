# MonkeyPox-project

Scope :- To better understand the statistics of the new Virus- MonkeyPox

Data set:- https://www.kaggle.com/datasets/deepcontractor/monkeypox-dataset-daily-updated

Imported CSV file into Postgresql and did a series of quries to solve the following questions

1. How many countries in the world has confired Monkeypx cases?
2. Number of contries with average Monkeypox cases?
3. The countries in the world which have more than average Monkeypox in the world?
4. What is the comparing ratio between suspected and confirmed cases by country?
5. Total number of people hopitalised,isolated,travel history in the World?
6. What are the unique symtoms for Monkeypox?
